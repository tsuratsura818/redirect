### 概要

QRコード・NFCタグ管理サービス「**Pivolink（ピボリンク）**」（URL: https://redirect.tsuratsura.com）は、本日よりベータ版の一般公開を開始いたします。

Pivolinkは、一度印刷したQRコードや店頭・商品に設置したNFCタグのリダイレクト先URLを、管理画面からいつでも変更できるSaaSサービスです。QRコードを刷り直さずに遷移先を変更できるため、チラシ・名刺・パッケージ・商品タグのコスト削減と運用効率化を実現します。

---

### 解決する課題

QRコードやNFCタグは一度印刷・設置すると「変更できない」ことが現場の大きな課題でした。

- キャンペーン終了後もQRコードが古いURLへ誘導してしまう
- リコール発生時にパッケージのQRコードをすぐに切り替えられない
- 売り切れ商品のNFCタグが404エラーに飛び続ける
- 毎月のイベントごとにQRチラシを刷り直すコストがかかる

Pivolinkはこれらすべての問題を、**既存のQRコード・NFCタグをそのまま使いながら**解決します。

---

### 主な機能

| 機能 | 概要 |
|------|------|
| **リダイレクト先の変更** | 管理画面からワンクリックでURL変更、即時反映 |
| **QRコード自動生成** | PNG/SVGで高品質なQRコードを自動生成 |
| **スケジュール切替** | 指定日時にリダイレクト先を自動切替 |
| **デバイス別振分** | iOS/Android/PCで異なるURLへ振り分け |
| **A/Bテスト** | トラフィックを分割し2URLの効果を比較 |
| **クッションページ** | リダイレクト前にお知らせ・クーポンを表示 |
| **アクセス解析** | スキャン数・デバイス比率をリアルタイム可視化 |
| **変更履歴** | いつ・誰が変更したか完全ログ管理 |

### 活用シーン

飲食店・小売・不動産・イベント・美容サロン・製造業・ECショップ・フィットネスジム・出版社・教育・ホテルなど、QRコード・NFCタグを活用するあらゆる業種に対応しています。
