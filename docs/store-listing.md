### キーワードフィールド

### カテゴリ
- プライマリ: ビジネス
- セカンダリ: ユーティリティ

### 説明文
【QRコード・NFCタグの飛び先を、いつでも自由に変更】

Pivolinkは、印刷済みのQRコードや設置済みのNFCタグのリダイレクト先を、
アプリからワンタップで変更できる管理ツールです。

もうQRコードを刷り直す必要はありません。

■ こんな場面で活躍します
・飲食店：テーブルのQRを季節メニューに差し替え
・イベント：告知→当日スケジュール→アーカイブを自動切替
・名刺：転職・異動してもURLそのまま、飛び先だけ更新
・店舗：キャンペーン期間だけ特設ページへ、終了後は通常ページに

■ アプリならではの機能
・カメラでQRスキャン → その場で飛び先を編集
・NFCタグに書き込み → タグにかざすだけの2ステップ
・NFCタグ読み取り → Pivolink管理タグなら即編集画面へ
・プッシュ通知 → アクセス急増やURL異常を即座にお知らせ
・ワンタップ共有 → QR画像をLINE・メール・SNSに送信

■ Web版との連携
PCのWeb版で設定を作り込み、現場ではアプリで操作。
同じアカウントでシームレスに使えます。

■ 料金
アプリのダウンロード・基本機能は無料です。

■ 対応
・iOS 15以降
・NFC機能はiPhone 7以降で利用可能

### App Review Notes
This app is a companion tool for the Pivolink SaaS web service (https://redirect.tsuratsura.com).
It provides native-only features that cannot be replicated in a web browser:

1. NFC Tag Read/Write - Uses Core NFC to read and write NDEF URI records to NFC tags
2. Camera QR Scanner - Real-time QR code scanning with jsQR library
3. Push Notifications - Firebase Cloud Messaging for access spike alerts
4. Native Share Sheet - iOS share extensions for QR code images

Demo account for testing:
- Email: review@pivolink.com
- Password: (作成時に設定)

The app does not contain any in-app purchases or links to external payment systems.
All billing is handled exclusively on the web version.

### 詳しい説明
【印刷済みQRコード & NFCタグの飛び先を、いつでも変更】

Pivolinkは、QRコードやNFCタグのリダイレクト先をクラウドで管理する
ビジネスツールです。一度印刷・設置したQRコードを刷り直すことなく、
飛び先URLをいつでも変更できます。

━━━━━━━━━━━━━━━━━
✦ 主な機能
━━━━━━━━━━━━━━━━━

◆ QRコードスキャン
カメラでQRを読み取り → Pivolinkのリンクならその場で編集画面へ

◆ NFCタグ読み書き
リンクを選んでタグにかざすだけ。NFCタグへの書き込みが2ステップで完了。
タグの中身は固定、飛び先だけ自由に変わる — これがPivolinkの価値です。

◆ アクセス解析
各QRコード・NFCタグのスキャン回数、デバイス、時間帯を確認

◆ プッシュ通知
アクセス数の急増、リダイレクト先のダウンを即座にお知らせ

◆ ワンタップ共有
QRコード画像をLINE・メール・SNSにネイティブ共有

━━━━━━━━━━━━━━━━━
